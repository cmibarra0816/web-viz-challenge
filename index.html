Summary: Latitude vs. X

What's the weather like as we approach the equator?

The purpose of this project was to analyze how weather changes. To accomplish this analysis, data was pulled from the OpenWeatherMap API - a dataset on over 500 cities was assembled. To visualize the dataset, Matplotlib was used to plot various aspects of the weather vs. latitude.

Factors: Temperature (F), Cloudiness (%), Wind Speed (mph), and Humidity (%).

This site provides the data, comparison, & visualizations created.
